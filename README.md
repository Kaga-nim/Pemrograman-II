## 🚀 Cara Menjalankan dengan NetBeans

1. Clone atau download repository ini
2. Buka project menggunakan NetBeans
3. Klik kanan pada project → **Run Project**

---

## 💻 Cara Menjalankan Tanpa NetBeans

1. Pastikan sudah menginstall Java (JDK) di komputer
   👉 Download JDK dari situs resmi: https://www.oracle.com/java/technologies/downloads/
2. Masuk ke folder project yang memiliki file `.jar` (biasanya di dalam folder `dist`)
3. Jalankan perintah berikut di Command Prompt / Terminal:

```
java -jar nama_project.jar
```

---

## ⚠️ Catatan

* Pastikan Java sudah terinstall dan terkonfigurasi di PATH sebelum menjalankan project
